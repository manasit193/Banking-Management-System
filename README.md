# Banking Management System

## Tech Stack
- Next.js
- TypeScript
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## Features
- User Authentication
- KYC Verification
- Admin Dashboard
- Transaction Management
- Monthly Reports
- Notifications
- Profile Management

## Installation

### Client
cd client
npm install
npm run dev

### Server
cd server
npm install
npm start
